# Markdown
#markdown #obsidian 
#### Six most important keys for Obsidian & markdown
1. The link `[[]]`
2. The tag `#tag`
	when putting tags in notes you can search for them. 
	```
	ctrl + click on tag
	```
3. itallics `*italics*`
4. bold `**bold**`
5. Lists `-`, `1.`
6. headings `#`, `##`, `###`, `####`, `#####`, `######`
